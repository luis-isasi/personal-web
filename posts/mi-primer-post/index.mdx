## ¿Cómo crear un proyecto de react con create-react-app?
```bash
  npx create-react-app my-app
```

## ¿Cómo crear un componente en react?
Creamos un archivo con el nombre de tu componente(recuerda que los nombres de los componentes deben iniciar en mayúscula),
dentro del archivo escribimos el siguiente código.

```bash
const MyComponent = () => {
  const text = "Hello World!";

  return (
    <div>
      <span>My first component</span>
      <span>{text}</span>
    </div>
  );
};

export default MyComponent;
```
